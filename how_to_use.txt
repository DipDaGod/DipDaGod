HOW TO USE — run all of this from the repo root (this folder).

SETUP (once)
  pip install -r scripts/requirements.txt
  Fill in scripts/config.py: GitHub username, name/role/education,
  languages, interests, links, ascii settings. Every script below reads
  from this one file.

QUICK START — generate everything
  python scripts/launch.py
  Checks config.py for unfilled placeholders (asks before continuing if
  it finds any), then runs all three steps below in order with a status
  box for each. Skips a step cleanly instead of failing if it doesn't
  apply yet (e.g. no source_photo).

INDIVIDUAL STEPS
  python scripts/make_info_card.py       -> info-card.svg
  python scripts/make_ascii_svg.py       -> ascii-img.svg
      Put a background-stripped photo in the repo root first, named
      "source_photo" (any extension, e.g. source_photo.png).
  python scripts/update_contributions.py -> contributions.svg
      + scripts/data/contributions.json
      Also runs on a schedule via .github/workflows/.

Or just edit the .py files directly and run them.
